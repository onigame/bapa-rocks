# BAPA Rocks - Manager
**BAPA Rocks** (also known as BAPA Manager) is a web application built on the **Yii 2 PHP Framework**. It manages the seasons, sessions, matches, and scoring for a pinball league.

## Documentation
For detailed system documentation, please refer to [PROJECT_DOCS.md](PROJECT_DOCS.md).

## Quick Start
1.  **Database**: Ensure the database is configured in `config/db.php`.
2.  **Dependencies**: Run `composer install` to install PHP dependencies.
3.  **Web Server**: Point your web server root to the `web/` directory.

## Testing
Run the test suite with:
```bash
vendor/bin/codecept run
```
